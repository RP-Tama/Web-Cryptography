Penggunaan

Web ini digunakan untuk mengenkripsi file dengan kata sandi tertentu dan juga bisa digunakan untuk mendekripsi file yang telah dienkripsi dengan kata sandi yang sama. Pastikan kata sandi yang digunakan untuk mengenkripsi file sama dengan yang digunakan untuk mendekripsi nantinya. Jika kata sandi hilang atau terlupa, maka tidak dapat dipulihkan kembali!

Kriptografi

Semua kriptografi dilakukan di sisi klien menggunakan Web Crypto API. File dienkripsi menggunakan enkripsi simetris AES-CBC 256-bit. Kunci enkripsi diperoleh dari kata sandi dan salt acak yang digunakan dalam PBKDF2 derivation dengan 10.000 iterasi hash SHA256.

Kompatibilitas dengan OpenSSL

Enkripsi yang digunakan oleh website ini kompatibel dengan OpenSSL. File yang dienkripsi menggunakan website ini dapat didekripsi menggunakan OpenSSL dengan perintah berikut:
openssl aes-256-cbc -d -salt -pbkdf2 -iter 10000 -in namafileterenkripsi -out namafileasli
File yang dienkripsi menggunakan OpenSSL dapat didekripsi menggunakan website ini dengan cara berikut:
openssl aes-256-cbc -e -salt -pbkdf2 -iter 10000 -in namafileasli -out namafileterenkripsi

Operasi dan Privasi

Website ini di-deploy menggunakan Vercel, sebuah platform pengembangan perangkat lunak yang terpercaya dan aman. Vercel menyediakan fitur keamanan dan enkripsi yang kuat untuk melindungi aplikasi dan konten yang di-hosting di platform mereka. Source code dari website ini dapat dilihat di GitHub melalui tautan berikut: https://github.com/RP-Tama/Web-Cryptography.
