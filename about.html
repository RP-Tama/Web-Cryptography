Penggunaan

Halaman web ini digunakan untuk mengenkripsi file dengan kata sandi tertentu dan juga bisa digunakan untuk mendekripsi file yang telah dienkripsi dengan kata sandi yang sama. Pastikan kata sandi yang digunakan untuk mengenkripsi file sama dengan yang digunakan untuk mendekripsi nantinya. Jika kata sandi hilang atau terlupa, maka tidak dapat dipulihkan kembali!

Operasi dan Privasi

Halaman ini menggunakan javascript yang berjalan dalam browser Anda untuk melakukan enkripsi dan dekripsi file di sisi client. Halaman ini tidak melakukan koneksi jaringan selama proses tersebut, sehingga file dan kunci Anda tidak pernah meninggalkan browser selama proses. Hal ini dapat diverifikasi secara independen dengan meninjau kode sumber atau dengan memonitor aktivitas jaringan browser Anda selama pengoperasian halaman ini. Halaman ini juga dapat diunduh dan dijalankan secara offline pada sistem Anda.

Kriptografi

Semua kriptografi dilakukan di sisi client menggunakan Web Crypto API. File dienkripsi menggunakan enkripsi simetris AES-CBC 256-bit. Kunci enkripsi diperoleh dari kata sandi dan salt acak yang digunakan dalam PBKDF2 derivation dengan 10000 iterasi hash SHA256.

Kompatibilitas dengan openssl

Enkripsi yang digunakan oleh halaman ini kompatibel dengan openssl.
File yang dienkripsi menggunakan halaman ini dapat didekripsi menggunakan openssl dengan perintah berikut:
openssl aes-256-cbc -d -salt -pbkdf2 -iter 10000 -in namafileterenkripsi -out namafileasli
File yang dienkripsi menggunakan openssl dapat didekripsi menggunakan halaman ini dengan cara berikut:
openssl aes-256-cbc -e -salt -pbkdf2 -iter 10000 -in namafileasli -out namafileterenkripsi

Mengoperasikan halaman ini secara offline

Halaman web ini mandiri. Halaman tidak memerlukan file pendukung apa pun; semua javascript dan css untuk halaman ini terdapat dalam kode sumber halaman ini. Untuk menjalankan halaman ini secara lokal di sistem Anda secara offline, cukup simpan halaman ini ke sistem Anda sebagai file .html, lalu buka file dari sistem Anda di browser web Anda (opsional dengan networking dinonaktifkan).
