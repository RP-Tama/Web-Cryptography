Penggunaan

Web ini digunakan untuk mengenkripsi file dengan kata sandi tertentu dan juga bisa digunakan untuk mendekripsi file yang telah dienkripsi dengan kata sandi yang sama. Pastikan kata sandi yang digunakan untuk mengenkripsi file sama dengan yang digunakan untuk mendekripsi nantinya. Jika kata sandi hilang atau terlupa, maka tidak dapat dipulihkan kembali!

Kriptografi

File dienkripsi menggunakan enkripsi simetris AES-CBC 256-bit. Kunci enkripsi diperoleh dari kata sandi dan salt acak yang digunakan dalam PBKDF2 derivation dengan 10000 iterasi hash SHA256.

Keamanan dan Privasi

Website ini di-deploy menggunakan Vercel, sebuah platform pengembangan perangkat lunak yang terpercaya dan aman. Vercel menyediakan fitur keamanan dan enkripsi yang kuat untuk melindungi aplikasi dan konten yang di-hosting di platform mereka. Source code dari website ini dapat dilihat di GitHub melalui tautan berikut: https://github.com/RP-Tama/Web-Cryptography.
